# AGENTS.md — よだかの星 手打ちうどん(引き継ぎ書)

架空のうどん店「よだかの星」を題材にした店舗サイト制作例。GitHub Pagesで公開中だが、実店舗の営業・予約窓口ではない。
このファイルは作業を引き継ぐエージェント向けの起点。**まず本ファイル → 次に `README.md` を読むこと。**

## プロジェクト概要
- 静的サイト(HTML/CSS/JS のみ・ビルド不要・依存ライブラリなし)
- 本体は `site/` 配下。トップ（星読みの盤＋全12品のお品書き）+ 下層3ページ（こだわり／店舗案内（架空）／制作ノート）+ 旧URL互換の`/menu/`転送ページ + 非導線のサンクスページ
- 公開URL: `https://techuueda-bot.github.io/yodaka-no-hoshi-udon/`
- 電話予約・フォーム送信・実在地点の地図は停止中。架空の番号に `tel:` を付けない
- デザイン方向: 「活字と丼 — 老舗×文学」。宮沢賢治「よだかの星」の世界観を、京都の仕出し店サイト「菱岩」(https://hishiiwa.com)の思想で表現

## ★ 触ってはいけない設計上の制約(過去の議論・施主フィードバックで確定)
1. **具象イラスト禁止**。料理・人・建物・地図を「絵」で描かない。実物の代理をベクターで描くとクリップアート化し、施主に却下された。許されるSVGは意匠のみ(星図・罫線・印章・一筆書きの線画・家紋・紙グレイン)。写真素材は無い前提で、タイポグラフィと余白で語る。
2. **実写真は未使用**。トップの全12品には、「イメージ写真（生成）」を用意している。季節限定3品は「星読みの盤」の月タブと星、定番・サイド9品は料理行から写真を開く。実店舗の料理写真として扱わず、一覧と写真ビューアの双方で生成表記をする。将来、権利確認済みの実写真が用意できたら、料理ごとに利用許諾とメニュー内容を確認して差し替える。想定位置と品質条件はREADME参照。
3. **ファーストビューを遅らせる演出は禁止(0秒)**。ローディング/スプラッシュの類は入れない。→ 施主も一度検討し「入れない」と決定済み。
4. **常時動作するアニメは増やさない**。動きは「視界に入った時」または「月・写真を選んだ時」に一度だけ、1.2s以内。マーキー・自動スライダー・カスタムカーソル・スクロール視差は入れない。
5. **色はトークンの範囲内**(生成り #f7f2e6 / 墨 #262622 / 濃紺 #1f2d3d / 金茶 #b8863f / 薄罫 #ddd0af / muted #766f5f)。塗り面・グラデ・ピル形状・ベタ塗りボタン禁止。金茶は小さな記号・ラベルのみ。
6. **時間で自動的に変わる事実の表示禁止**(JSの曜日判定で「営業中」等を出さない。祝日・臨時休業で嘘になる)。フッターは架空設定の営業時間と「営業・予約受付はありません」を静的表示する。
7. **すべての動きは `motion-stop` ボタン(と reduced-motion)で停止できること**。新しい動きを足したら停止ポリシーに接続する。
8. デザイン基盤は `assets/css/style.css`(トークン→レイアウト→コンポーネント→モーション→停止の順)と `assets/js/main.js`(IntersectionObserver + CSS transition のみ。ライブラリ追加しない)。

## 実装済み(要約。詳細は README.md)
- タイポグラフィ主導のエディトリアル。濃紺の「星読みの盤」、巨大一文字の背景、手彫り印章ロゴ、紙グレイン
- 季節限定3品は月タブと既存の星図SVGを一つの選択装置として使用。選択中の一杯だけを共有3:2写真と縦書きで見せる
- 定番7品・サイド2品はトップのお品書きに統合。料理行から共有ビューアを開き、上端の「前の一杯／写真を閉じる／次の一杯」で同じグループ内を移動する。閉じた後は選んだ献立行へフォーカスを戻す
- `/menu/`は重複画面を持たず、トップの`/#oshinagaki`へ転送する互換ページ
- 地図未掲載表示、簡潔な共通フッター、下層の巨大一文字、漢数字の献番、罫線描画、morelinkホバーを維持
- 全6HTMLで「架空店舗サイト／制作例」を明示。OGP・canonical・構造化データは公開URL基準
- 主ナビは「トップ / お品書き / こだわり」の3項目。制作情報はフッターの「制作ノート」、架空店舗情報は「店舗案内（架空）」へ分離
