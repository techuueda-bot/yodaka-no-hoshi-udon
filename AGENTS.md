架空のうどん店「よだかの星」の店舗サイト(実務モード=来店・電話予約が目的)。
このファイルは作業を引き継ぐエージェント向けの起点。**まず本ファイル → 次に `README.md` を読むこと。**

## プロジェクト概要
- 静的サイト(HTML/CSS/JS のみ・ビルド不要・依存ライブラリなし)
- 本体は `site/` 配下。トップ + 下層4ページ(メニュー/こだわり/店舗案内/お問い合わせ)+ サンクスページ
- デザイン方向: 「活字と丼 — 老舗×文学」。宮沢賢治「よだかの星」の世界観を、京都の仕出し店サイト「菱岩」(https://hishiiwa.com)の思想で表現

## ★ 触ってはいけない設計上の制約(過去の議論・施主フィードバックで確定)
1. **具象イラスト禁止**。料理・人・建物・地図を「絵」で描かない。実物の代理をベクターで描くとクリップアート化し、施主に却下された。許されるSVGは意匠のみ(星図・罫線・印章・一筆書きの線画・家紋・紙グレイン)。写真素材は無い前提で、タイポグラフィと余白で語る。
2. **写真は未使用**。将来実写真が用意できたら差し替え可だが、無い状態で成立する設計を保つ。
3. **ファーストビューを遅らせる演出は禁止(0秒)**。ローディング/スプラッシュの類は入れない。→ 施主も一度検討し「入れない」と決定済み。
4. **常時動作するアニメは「ヒーローの星の明滅」1つだけ**。他の動きはすべて「視界に入った時に一度きり・1.2s以内」。マーキー・自動スライダー・カスタムカーソル・スクロール視差は入れない。
5. **色はトークンの範囲内**(生成り #f7f2e6 / 墨 #262622 / 濃紺 #1f2d3d / 金茶 #b8863f / 薄罫 #ddd0af / muted #766f5f)。塗り面・グラデ・ピル形状・ベタ塗りボタン禁止。金茶は小さな記号・ラベルのみ。
6. **時間で自動的に変わる事実の表示禁止**(JSの曜日判定で「営業中」等を出さない。祝日・臨時休業で嘘になる)。曜日帯は静的表示+「火曜定休」テキスト併記。
7. **すべての動きは `motion-stop` ボタン(と reduced-motion)で停止できること**。新しい動きを足したら停止ポリシーに接続する。
8. デザイン基盤は `assets/css/style.css`(トークン→レイアウト→コンポーネント→モーション→停止の順)と `assets/js/main.js`(IntersectionObserver + CSS transition のみ。ライブラリ追加しない)。
